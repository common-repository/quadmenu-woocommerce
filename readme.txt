Create a Mega Menu in WooCommerce.

== Description ==
[Mega Menu](http://quadmenu.com/) | [WooCommerce Mega Menu](http://quadmenu.com/storefront/)

This WordPress Plugin allow you to create a Mega Menu in WooCommerce and integrates QuadMenu with the WooCommerce theme settings. Requires QuadMenu and WooCommerce.

== Screenshots ==

Demo Site: http://quadmenu.com/storefront/

1. QuadMenu Mega Menu working with WooCommerce
2. This plugin adds a preconfigured "WooCommerce" theme to fit QuadMenu Mega Menu settings.

== Installation ==

1. Go to the Plugins page in WordPress
1. Search for "WooCommerce Mega Menu"
1. Click "Install"

After installation go to Appearance > Menus and create your mega menu.
